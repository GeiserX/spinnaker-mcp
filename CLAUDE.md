# CLAUDE.md — Spinnaker-MCP

## Overview
MCP (Model Context Protocol) server that exposes any Spinnaker instance as an MCP v1 server via the Gate API. Provides tools for managing applications, pipelines, executions, infrastructure, and deployment strategies.

## Tech Stack
- Go 1.25
- MCP v1 protocol (stdio and SSE transports)
- Docker (image: `drumsergio/spinnaker-mcp`)
- npm package (`spinnaker-mcp`) for npx usage
- GoReleaser for builds
- Helm chart for Kubernetes deployment

## Development
```bash
go build ./cmd/server
go test ./...

# Run locally
export SPINNAKER_URL="https://your-spinnaker-gate"
go run ./cmd/server
```

## Architecture
- `cmd/server/` — server entrypoint
- `internal/tools/` — MCP tool implementations (applications, pipelines, executions, etc.)
- `internal/toolsets/` — tool grouping and registration
- `internal/resources/` — MCP resource definitions
- `internal/prompts/` — MCP prompt templates
- `client/` — JavaScript client wrapper for npx
- `config/` — configuration handling
- `helm/` — Helm chart for Kubernetes deployment
- `glama.json` — Glama MCP registry metadata

## Key Rules
- Never hardcode Spinnaker credentials; use environment variables or config files
- Published to npm, Docker Hub, Official MCP Registry, Glama, and mcp.so
