## Project Overview

MCP Server for Spinnaker — application management, pipeline operations, and deployment infrastructure monitoring via MCP.

- **Repo:** [GeiserX/spinnaker-mcp](https://github.com/GeiserX/spinnaker-mcp)
- **Language:** Go
- **License:** GPL-3.0
- **SDK:** [mark3labs/mcp-go](https://github.com/mark3labs/mcp-go)

## Architecture

MCP (Model Context Protocol) server built in Go. Wraps the Spinnaker Gate REST API, exposing 13 tools for AI assistants to manage Spinnaker applications, pipelines, executions, and infrastructure.

### Key Directories

- `cmd/server/` — Entry point, MCP server initialization and tool registration
- `client/` — Gate API HTTP client with multi-auth support (bearer, basic, x509)
- `config/` — Environment-based configuration loader
- `internal/tools/` — MCP tool implementations (one file per tool)
- `version/` — Build-time version injection
